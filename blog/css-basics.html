CSS Basics: First Steps in Web Design

May 22, 2025 12 min read CSS Design

If you built the skeleton of the website with HTML, now it is time to dress it up with CSS (Cascading Style Sheets). CSS handles all styling, from simple tasks like changing text colors to complex designs like setting the layout and adding animation effects.

1. CSS Syntax Structure

CSS is a language that defines Who (Selector) to change and How (Property). The basic syntax is as follows:

h1 {
  color: blue;
  font-size: 24px;
}
  • Selector: Selects the target to apply the style to. In the example above, the h1 tag is the selector.
  • Property: The type of style you want to change (e.g., color, font-size).
  • Value: The specific value to apply to the property (e.g., blue, 24px).

2. 3 Ways to Apply CSS

(1) Inline Style

Using the style attribute directly inside HTML tags. It's good for quick use, but not recommended as it clutters the code.

<h1 style="color: red;">Red Heading</h1>

(2) Internal Style Sheet

Writing a <style> tag inside the <head> of the HTML document. Useful when making simple pages.

(3) External Style Sheet - Recommended!

Creating a separate file with a .css extension and linking it. It is easiest to manage as styles can be shared across multiple pages.

<link rel="stylesheet" href="style.css">

3. Essential CSS Properties

Color and Background

  • color: Changes text color.
  • background-color: Changes background color.

Font and Text

  • font-size: Adjusts text size (using px, rem, etc.).
  • font-weight: Adjusts text thickness (bold, 400, 700, etc.).
  • text-align: Aligns text (left, center, right).

Box Model - Very Important!

All elements on the web are made of rectangular boxes. These are the properties that handle these boxes.

  • width / height: Width and Height
  • margin: Outer spacing of the box
  • padding: Inner spacing of the box
  • border: Border line

4. Practice: Making My Own Button

Enter the following code into the editor to create a pretty button!

.my-btn {
  background-color: #2563eb;
  color: white;
  padding: 10px 20px;
  border: none;
  border-radius: 5px;
  cursor: pointer;
  font-size: 16px;
}
.my-btn:hover {
  background-color: #1d4ed8;
}

Conclusion

CSS is a language that has depth the more you learn it. Start with changing colors and sizes, and gradually expand to layout (Flexbox, Grid) and Responsive Web Design. Your website will transform as if touched by a professional.
