HTML Basics: Perfect Guide for Beginners

May 20, 2025 10 min read HTML Study

Welcome to the world of web development! If you want to build a website, the first language you must learn is HTML (HyperText Markup Language). In this article, we will delve deep into what HTML is, how it is structured, and the essential tags you must know.

1. What is HTML?

HTML is the markup language responsible for the Structure of a web page. Just as we erect a steel frame first when building a building, we need to frame it with HTML when building a website. CSS is for interior design (styling) on that frame, and JavaScript gives function (behavior) like turning on lights or opening windows.

"HTML is not a programming language, but a markup language that tells the browser how content should be displayed."

2. Basic Structure of an HTML Document

All HTML documents follow standard structures like the following. Copy and paste this code into the editor.

<!DOCTYPE html>
<html lang="en">
  <head>
    <meta charset="UTF-8">
    <title>My First Website</title>
  </head>
  <body>
    <h1>Hello!</h1>
    <p>This is the body content.</p>
  </body>
</html>

Key Components Description

  • <!DOCTYPE html>: Informs the browser that this document follows the latest HTML5 standards.
  • <html>: The top-level element that wraps the entire page.
  • <head>: Contains information for search engines or browsers (metadata, title, stylesheet links, etc.) that are not visible on the screen.
  • <body>: This is where text, images, buttons, etc. actually shown to the user go.

3. 5 Essential Tags You Must Know

There are numerous tags in HTML, but the following 5 are most frequently used, so you must master them.

(1) Heading Tags (<h1> ~ <h6>)

Represents a Heading. <h1> is the largest heading, and the smaller the number, the smaller the heading. Search engines recognize the <h1> tag as the core topic of the page, so it is recommended to use it only once per page.

(2) Paragraph Tag (<p>)

Used to create text paragraphs. It is important to divide paragraphs appropriately for readability when writing long texts.

(3) Link Tag (<a>)

Creates a hyperlink (Anchor) to move to another page.

<a href="https://google.com" target="_blank">Go to Google</a>

(4) Image Tag (<img>)

Inserts an image. Put the image address in the src attribute and the image description in the alt attribute.

<img src="cat.jpg" alt="Cute Cat Photo">

(5) List Tags (<ul>, <ol>, <li>)

Unordered lists use <ul>, ordered lists use <ol>, and each item is wrapped in <li>.

4. Importance of Semantic Markup

In modern web development, it is important to use tags that fit the Meaning (Semantic), not just how they look. For example, using <header> for headers, <main> for the body, and <footer> for the footer instead of just using <div> is good for Search Engine Optimization (SEO) and improves code readability.

Conclusion

You have now laid the foundation for HTML. The next step is to learn CSS to decorate your site beautifully. Use our EasyHTML to practice the tags you learned today! Seeing is believing; typing the code yourself is the best way to learn.
