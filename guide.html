사용 가이드

에디터를 효과적으로 활용하는 방법을 단계별로 안내합니다.

1. 실시간 미리보기

에디터에 입력한 HTML은 오른쪽 패널에서 즉시 렌더링됩니다. ‘미리보기 새로고침’ 버튼으로 수동 갱신도 가능합니다.

2. 코드 포맷팅

난잡한 HTML을 정리하려면 상단 ‘포맷팅’ 버튼을 누르세요. 자동으로 들여쓰기와 줄바꿈이 정리됩니다.

3. 샘플 템플릿

처음 시작하기 어렵다면 ‘샘플’ 버튼을 눌러 예제 코드로 시작하세요. 학습과 테스트에 유용합니다.

4. 파일 관리

  • 로컬 저장/불러오기: 브라우저에 임시 저장하여 나중에 이어서 작업합니다.
  • 파일 업로드: 기존 HTML 파일을 불러와 편집할 수 있습니다.
  • 다운로드: 완성된 결과를 파일로 저장합니다.

5. 단축키

  • Alt+1: 새 탭 미리보기
  • Alt+2: 포맷팅
  • Alt+3: 내용 지우기
  • Alt+4/5: 저장/불러오기
  • Alt+6/7: 업로드/다운로드
  • Alt+8/9: 샘플/전체화면
  • Ctrl+S: 저장, F11: 전체화면
