CSS Flexbox 완벽 가이드: 쉽고 빠른 레이아웃

2025년 6월 10일 7분 읽기 디자인

CSS로 레이아웃을 잡는 일은 과거에 float와 clearfix 때문에 악몽과 같았습니다. 그러다 Flexbox가 등장했죠. 오늘날 이것은 현대적이고 반응형인 레이아웃을 손쉽게 구축하기 위한 표준입니다. 이 가이드에서 Flexbox의 핵심을 짚어보겠습니다.

Flexbox란 무엇인가요?

Flexible Box Layout Module, 즉 Flexbox는 요소를 행(row)이나 열(column)로 배치하기 위한 1차원 레이아웃 방식입니다. 요소의 정확한 크기를 몰라도 가용한 공간을 채우고 요소를 완벽하게 정렬하는 데 탁월합니다.

시작하기: 부모 요소 (Flex Container)

Flexbox를 사용하려면 부모 요소에 display: flex;를 적용하기만 하면 됩니다. 이렇게 하면 부모는 flex 컨테이너가 되고, 그 직계 자식들은 flex 아이템이 됩니다.

주요 부모 속성:

  • flex-direction: 방향을 설정합니다. row(기본값) 또는 column.
  • justify-content: 주축(보통 가로)을 따라 요소를 정렬합니다. center, space-between, flex-start 등이 주로 쓰입니다.
  • align-items: 교차축(보통 세로)을 따라 요소를 정렬합니다. 요소를 상하 중앙에 배치할 때 유용합니다.
  • flex-wrap: 공간이 부족할 때 요소가 다음 줄로 넘어가게 할지 설정합니다.

자식 요소 (Flex Items)

컨테이너 안의 개별 아이템도 제어할 수 있습니다.

  • flex-grow: 아이템이 필요한 경우 얼마나 더 커질 수 있는지 정의합니다.
  • order: HTML 구조를 바꾸지 않고도 요소가 나타나는 순서를 변경할 수 있습니다.

실용 예제: 모든 것을 중앙으로

CSS의 "성배"와 같았던 'div 완벽하게 중앙 정렬하기'는 Flexbox를 쓰면 단 3줄로 끝납니다.

.container {
  display: flex;
  justify-content: center;
  align-items: center;
  height: 100vh;
}

결론

Flexbox는 모든 웹 개발자가 마스터해야 하는 강력한 도구입니다. 복잡한 레이아웃에 필요한 코드량을 획기적으로 줄여주며, 어떤 화면 크기에서도 사이트가 멋지게 보이도록 보장합니다. 오늘 우리 에디터에서 Flexbox를 직접 실습해 보세요!
