HTML 기초: 초보자를 위한 완벽 입문 가이드

2025년 5월 20일 10분 읽기 HTML 공부

웹 개발의 세계에 오신 것을 환영합니다! 웹사이트를 만들고 싶다면 가장 먼저 배워야 할 언어가 바로 HTML (HyperText Markup Language)입니다. 이 글에서는 HTML이 무엇인지, 어떻게 구성되는지, 그리고 꼭 알아야 할 필수 태그들에 대해 심층적으로 다뤄보겠습니다.

1. HTML이란 무엇인가요?

HTML은 웹 페이지의 구조(Structure)를 담당하는 마크업 언어입니다. 우리가 건물을 지을 때 철골 구조를 먼저 올리듯이, 웹사이트를 만들 때도 HTML로 뼈대를 잡아야 합니다. CSS는 그 뼈대 위에 인테리어(디자인)를 하는 것이고, JavaScript는 전등을 켜거나 창문을 여는 것과 같은 기능(동작)을 부여하는 것입니다.

"HTML은 프로그래밍 언어가 아니라, 콘텐츠가 어떻게 보여져야 하는지 브라우저에게 알려주는 마크업 언어입니다."

2. HTML 문서의 기본 구조

모든 HTML 문서는 다음과 같은 표준 구조를 따릅니다. 에디터에 이 코드를 복사해서 붙여넣어 보세요.

<!DOCTYPE html>
<html lang="ko">
  <head>
    <meta charset="UTF-8">
    <title>나의 첫 웹사이트</title>
  </head>
  <body>
    <h1>안녕하세요!</h1>
    <p>이것은 본문입니다.</p>
  </body>
</html>

주요 구성 요소 설명

  • <!DOCTYPE html>: 이 문서가 HTML5 최신 표준을 따른다는 것을 브라우저에 알립니다.
  • <html>: 전체 페이지를 감싸는 최상위 요소입니다.
  • <head>: 화면에는 보이지 않지만 검색 엔진이나 브라우저를 위한 정보(메타 데이터, 제목, 스타일시트 연결 등)를 담습니다.
  • <body>: 실제로 사용자에게 보여지는 텍스트, 이미지, 버튼 등이 들어가는 곳입니다.

3. 반드시 알아야 할 필수 태그 5가지

HTML에는 수많은 태그가 있지만, 다음 5가지는 가장 빈번하게 사용되므로 반드시 숙지해야 합니다.

(1) 제목 태그 (<h1> ~ <h6>)

제목(Heading)을 나타냅니다. <h1>이 가장 큰 제목이며, 숫자가 커질수록 작은 제목이 됩니다. 검색 엔진은 <h1> 태그를 페이지의 핵심 주제로 인식하므로, 페이지당 한 번만 사용하는 것이 좋습니다.

(2) 문단 태그 (<p>)

텍스트 단락(Paragraph)을 만들 때 사용합니다. 긴 글을 작성할 때 가독성을 위해 적절히 문단을 나누는 것이 중요합니다.

(3) 링크 태그 (<a>)

다른 페이지로 이동하는 하이퍼링크(Anchor)를 만듭니다.

<a href="https://google.com" target="_blank">구글로 이동</a>

(4) 이미지 태그 (<img>)

이미지를 삽입합니다. src 속성에 이미지 주소를, alt 속성에 이미지 설명을 넣습니다.

<img src="cat.jpg" alt="귀여운 고양이 사진">

(5) 목록 태그 (<ul>, <ol>, <li>)

순서가 없는 목록은 <ul>, 순서가 있는 목록은 <ol>을 사용하며, 각 항목은 <li>로 감쌉니다.

4. 시멘틱(Semantic) 마크업의 중요성

요즘 웹 개발에서는 단순히 보여지는 것뿐만 아니라, 의미(Semantic)에 맞는 태그를 사용하는 것이 중요합니다. 예를 들어, <div>만 남발하는 것보다, 헤더에는 <header>, 본문에는 <main>, 하단에는 <footer> 태그를 사용하는 것이 검색 엔진 최적화(SEO)에도 좋고 코드의 가독성도 높여줍니다.
