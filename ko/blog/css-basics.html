CSS 시작하기: 웹사이트 디자인의 첫걸음

2025년 5월 22일 12분 읽기 CSS 디자인

HTML로 웹사이트의 뼈대를 만들었다면, 이제 CSS (Cascading Style Sheets)로 옷을 입힐 차례입니다. CSS는 글자 색상을 바꾸는 단순한 작업부터, 화면 배치(Layout)를 잡고 애니메이션 효과를 주는 복잡한 디자인까지 모든 스타일링을 담당합니다.

1. CSS 문법 구조

CSS는 누구를(선택자), 어떻게(속성) 바꿀지를 정의하는 언어입니다. 기본 문법은 다음과 같습니다.

h1 {
  color: blue;
  font-size: 24px;
}
  • 선택자 (Selector): 스타일을 적용할 대상을 선택합니다. 위 예제에서는 h1 태그가 선택자입니다.
  • 속성 (Property): 변경하고 싶은 스타일의 종류입니다. (예: color, font-size)
  • 값 (Value): 속성에 적용할 구체적인 값입니다. (예: blue, 24px)

2. CSS를 적용하는 3가지 방법

(1) 인라인 스타일 (Inline Style)

HTML 태그 안에 직접 style 속성을 사용하는 방법입니다. 급할 때 쓰기 좋지만, 코드가 지저분해지므로 추천하지 않습니다.

<h1 style="color: red;">빨간색 제목</h1>

(2) 내부 스타일시트 (Internal Style Sheet)

HTML 문서의 <head> 안에 <style> 태그를 만들어 작성하는 방법입니다. 간단한 페이지를 만들 때 유용합니다.

(3) 외부 스타일시트 (External Style Sheet) - 권장!

확장자가 .css인 별도의 파일을 만들고 링크를 거는 방법입니다. 여러 페이지에서 스타일을 공유할 수 있어 관리가 가장 쉽습니다.

<link rel="stylesheet" href="style.css">

3. 필수 CSS 속성 베스트

색상과 배경

  • color: 글자 색상을 변경합니다.
  • background-color: 배경 색상을 변경합니다.

폰트와 텍스트

  • font-size: 글자 크기를 조절합니다. (px, rem 등을 사용)
  • font-weight: 글자 굵기를 조절합니다. (bold, 400, 700 등)
  • text-align: 텍스트를 정렬합니다. (left, center, right)

박스 모델 (Box Model) - 매우 중요!

웹의 모든 요소는 사각형 박스로 이루어져 있습니다. 이 박스를 다루는 속성들입니다.

  • width / height: 너비와 높이
  • margin: 박스 바깥쪽 여백
  • padding: 박스 안쪽 여백
  • border: 테두리

4. 실습: 나만의 버튼 만들기

다음 코드를 에디터에 입력하여 예쁜 버튼을 만들어보세요!

.my-btn {
  background-color: #2563eb;
  color: white;
  padding: 10px 20px;
  border: none;
  border-radius: 5px;
  cursor: pointer;
  font-size: 16px;
}
.my-btn:hover {
  background-color: #1d4ed8;
}

마치며

CSS는 배우면 배울수록 깊이가 있는 언어입니다. 처음에는 색상과 크기를 바꾸는 것부터 시작해서, 점차 레이아웃(Flexbox, Grid)과 반응형 웹 디자인으로 영역을 넓혀가 보세요. 여러분의 웹사이트가 전문가의 손길이 닿은 것처럼 변할 것입니다.
